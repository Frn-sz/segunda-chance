## Implementação do algoritmo segunda chance em C++

Comando para compilação:
```bash
g++ -o main main.cpp
```
Ou
```bash
clang++ -o main main.cpp
```

### Argumentos

O programa pode receber até 2 argumentos

1. Delay de print em ms

O comando abaixo coloca o delay de 1s (1000ms) em cada passo do algoritmo.
```bash
./main 1000
```

2. Quantidade de páginas

O comando abaixo executa o algoritmo com 10 páginas e sem delay.
```bash
./main 0 5
```

### Ambos os comandos tem valores padrão de 0 e 3, respectivamente.
